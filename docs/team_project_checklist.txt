Team Project Demo checklist
For the demo, please include the following:

UI - Web or mobile based, integrated with API and DB
Seller role - Create and manage listings, Upload photos - View listings, listing details
All users - Search and filter by category and price, AI powered NL based search
In-app chat for negotiation -
Buyer role - send message for a listing;
Seller role - view and respond to messages for each listing they own
Seller - Mark items as sold
Buyers - Report incomplete listings to admin
Admin role - disable or delete listings
Cloud configuration page - API and Database running on AWS or another cloud provider in an Auto Scaled EC2 Cluster with Load Balancer
Display Javascript console on browser to show Network calls from UI to API running on the cloud
